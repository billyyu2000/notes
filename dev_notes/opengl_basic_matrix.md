## OpenGL Translate Matrix

[glTranslate](https://lmb.informatik.uni-freiburg.de/people/reisert/opengl/doc/glTranslate.html)

```
1 0 0 x
0 1 0 y
0 0 1 z
0 0 0 1
```

inverse of T 

```
1 0 0 -x
0 1 0 -y
0 0 1 -z
0 0 0 1
```

```python
trans: 1,2,3
[[1. 0. 0. 1.]
 [0. 1. 0. 2.]
 [0. 0. 1. 3.]
 [0. 0. 0. 1.]]
inv trans:
[[ 1.  0.  0. -1.]
 [ 0.  1.  0. -2.]
 [ 0.  0.  1. -3.]
 [ 0.  0.  0.  1.]]
```
