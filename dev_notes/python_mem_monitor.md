# Python 内存测试

- install `psutil`

```python
import gc 
import os 
import psutil

def test(): 
    for i in range(10000000):
        pass


if __name__ == '__main__' :
    gc.collect()
    test()
    p = psutil.Process(os.getpid())
    mem =  p.memory_info() 
    print mem ,  mem.rss / 1024/1024  , "M used"
```
