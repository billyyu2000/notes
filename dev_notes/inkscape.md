# Inkscape

## Interface and Basic Drawing

- move page : 
    - ctrl + right mouse button
    - or hold down middle mouse button
- scale UI:
    - ctrl + mouse whell
- set stroke color
    - shift + pick color
- select & scale a shape will also scale its stroke.
- circle tool
    - style: 1. pacman (outside)  2. half moon (partial circle) (inside)
